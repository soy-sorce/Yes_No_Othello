# Yes/No Othello ルール概要

このドキュメントは、8×8のオセロ盤を使う「Yes/No Othello」のルールを日本語でまとめたものです。基本的な挟み取りの仕組みは通常のオセロと同じですが、毎ターンに呼び出される yesno.wtf API の結果によって置ける石が変化します。

## 基本の準備
- 盤面は 8×8 のマスを使用。
- 黒石（YES）と白石（NO）の2色を使い、初期配置は通常のオセロと同じ（中央4マスに黒白が対角で2枚ずつ）。
- 駒の中央には大きく `YES` / `NO` の文字が描かれており、黒石には白文字 YES、白石には黒文字 NO が重なるので、自分の色が一目でわかる。
- プレイヤーは交互に手番を持ち、常に相手の石を1枚以上挟める場所にしか置けない。
- 盤が埋まるか、互いに連続でパスするまでプレイし、最終的に自分の石が多い方が勝者。枚数が同じなら引き分け。

## ターンの流れ
1. 手番の開始時に yesno.wtf API を呼び出し、`YES` / `NO` / `MAYBE` のいずれかを受け取る。通信に失敗した場合はランダムに YES か NO を代用。
2. 結果に応じて置ける石や効果が変わる。
3. プレイヤーはマウスで合法手を選び、その結果に従って石を置く。
4. 置ける場所がなければ自動的にパスし、相手に手番が移る。

## API 結果による挙動
- `YES`: 黒石を置く指示。手番側が黒なら通常通り石を置き、挟んだ相手石をひっくり返す。手番側が白なのに YES が出た場合は、相手色の石を置くことになるため挟み取りは発生しない（石は置かれるが裏返し不可）。
- `NO`: 白石版の YES と同じ扱い。白の手番なら通常通り、黒の手番なら相手色の石を置くだけで裏返せない。
- `MAYBE`: 自分の色の石を置き通常の挟み取りを行った後、置いた石に隣接する（8方向）相手石を強制的にすべて自分色に裏返す。効果発動中は盤面が一瞬点滅して知らせる。

## 終了条件と勝敗
- 両者とも合法手がなくなり連続でパスした場合、または盤面が埋まった時点で終了。
- 盤上の石数を比較し、多い方が勝ち。枚数が同じなら引き分け。

## 操作方法（デフォルト設定）
- 盤面のマスをクリックして石を置く。
- GIF ポップアップを ON にした場合、表示中にクリックまたはキーボードのいずれかを押すと盤面に戻れる。
- ウィンドウを閉じるとその時点でゲームが終了し、勝敗結果がターミナルへ表示される。
